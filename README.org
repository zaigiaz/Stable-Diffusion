* Overview

This is a repo for my stable-diffusion-1.5 experiments and with other
2021-2023 diffusion models to get them to intentionally create wierd, offputting art

- models used: https://huggingface.co/Jiali/stable-diffusion-1.5

- other models you could use: https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0

** How to setup

You need these python3 packages:
- torch 
- torchvision
- diffusers
- transformers
- accelerate
- safetensors

download the safetensors models from the hugging-face repo, but not the pytorch.bin models,
You will probably have to either use wget or another download tool to quickly get all the json and model files for each dir

** Directories Needed and files
- the main safetensors model
- model_index.json
- feature_extractor/
- scheduler/
- text_encoder/
- tokenizer/
- unet/
- vae/

You should always make sure to grab the safetensors models as they are the updated ones that have
faster serialization and are a more safer modern format
